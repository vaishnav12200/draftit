# Contributing to DraftIt

Thanks for taking the time to help. DraftIt aims to feel like a filmmaker's writing room: dark, immersive, calm, and focused. If a UI element does not help writing, it should not exist.

## Getting set up

1. Install dependencies: `pnpm install`
2. Run the dev server: `pnpm dev`
3. Optional checks: `pnpm check`

## How to contribute

- Start with an issue or discuss the scope in Discord.
- Create a branch: feature/..., fix/..., or chore/...
- Keep changes small and focused.
- For UI updates, include before/after screenshots (desktop width) and check mobile behavior.
- Follow the design system and typography rules from the README.

## Pull request checklist

- App runs locally without errors.
- `pnpm check` passes.
- Screenshots included for UI changes.
- Docs updated when needed.

## Reporting issues

Include:

- Steps to reproduce
- Expected vs actual behavior
- Screenshots or screen recordings if visual
- Browser and OS details
